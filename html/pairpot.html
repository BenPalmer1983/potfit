Central Pair Potentials

Short range pair potentials can be activated in potfit with the compile target PAIR.

Pair potentials can be given in tabulated form or with analytic parameters.

Tabulated Pair Potentials

For use in potfit, any potential needs to be specified at a limited number of sampling points. Position and value of these sampling points are given in the file specified with the parameter potfile. The format used in this file is described above. It is essential that the potential is tabulated equidistant in the the atom distance, not in the sqare (as it is done in IMD). Note that if the potential does not extend to radii small enough to cover all pair distances appearing in the configurations, the program will exit with an short distance error.

Analytic Pair Potentials

In order to use analytic potentials in potfit, you have to specify the functions and the values of the parameters in the potential file. The format used in this file is described above.
With analytic potentials there is also the option to add an additional energy term to the fit, the so called chemical potential.

Chemical Potentials

cp_Mg -1 -10 0
cp_Zn -1 -10 0
