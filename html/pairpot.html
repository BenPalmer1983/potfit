Central Pair Potentials

Short range pair potentials are active by default in potfit.

Pair potentials are given in tabulated form

Tabulated Pair Potentials

For use in potfit, any potential needs to be specified at a limited number of sampling points. Position and value of these sampling points are given in the file specified with the parameter potfile. The format used in this file is described above. It is essential that the potential is tabulated equidistant in the the atom distance, not in the sqare (as it is done in IMD). Note that if the potential does not extend down to r=0, the program will exit with an short distance error.
